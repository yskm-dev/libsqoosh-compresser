LibSqoosh-Compresser

# Overview

- [ ] フォルダの初期化
- [x] JPEG を mozjpeg で圧縮
- [x] PNG を mozjpeg で圧縮
- [x] JPEG,PNG を WebP に変換
- [x] その他のファイルはただコピーを作成

# Useage

## Install

```
npm install
```

## Run Sqoosh

```
npm run optimise
```
